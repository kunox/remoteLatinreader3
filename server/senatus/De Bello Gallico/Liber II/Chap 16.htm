Sabim以下伝聞。

non amplius milibus passuum X : ampliusは対格なので、milibusと奪格になっているのはおかしい！miliaというテキストも多い (ampliusは中性、単対格なので副詞との見方も成り立つが、時間空間の長さを表すのは対格、という原則からは対格であるべき？）

omnes Nervios consedisse trans id flumen:「全Nervii族が河向こう側で配置されている」

expectare adventum Romanorum : ローマ軍の到着を待つ。

finitimis suisは同格。finitimisはAtrebantibusなどと同じで奪格。suisは与格（NewLatin 192)

uti eandem belli fortunam : uti = ut 目的の節。

persuaserantが直説法なのはカエサルの直接の言葉であり、伝聞（間接話法）の中ではないからである。

utrisqueは指している2つが別々の場合、複数形をとる。（NewLatin355)

（カエサルは）その領土から３日間の旅程を行ったとき、捕虜から次の事を知った。Sabim河は自軍の陣営から１０マイル以上は離れていないこと、そしてその河を渡ったところでNevii族が、隣人であるAtrebates とVeromanduiと一緒にローマ軍の到着を待っている事。（と言うのはネウイ族はその二つの部族を説得して、同じ戦争で幸運を試すようにしたのである）
