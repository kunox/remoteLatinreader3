magis ut natura loci [deiectusque collis] et necessitas temporis (postulabata) quam ut ratio atque ordo militaris rei.

ratio atque ordo militaris rei : 軍事の規則と秩序 (ratio=theory, ordo=arrangement)

aliae alia in parte : some in one place , some in another place 慣用句。Aliusは別な格で繰り返され、または同じ語幹の副詞と共に繰り返され二つの文を簡潔に表す。（AG315 c)

opus est : 手段の奪格の特別な変種で一つ。Opus est 奪格で「～が必要だ」の意。奪格ではなく普通の名詞が主語になることもある。または受動の完了分詞が来ることもある。（NewLatin218.2)

Liddleによると二つの解釈が可能。１）opus est が述語で必要となるものが主語　nobis opus est dux.  2) opus estが主語で必要となるものが奪格。こちらが一般的　opus est gladio.(we need a sword) ここでは１）で、補給が必要だった。

構文的には特徴になる前置詞などがなくて解りにくい。

cum diversae legiones rersisterent hostibus, poterant neque subsidia conlocari, neque ~, neque ~ : が骨子。「分散された軍団が敵に対抗した時、（視界が悪かったので）色々なことができなかった、と言う事。

saepibus densissimis interiectis : 独立奪格。「厚い木々で出来た防護壁があったため」

prospectus impediretur : 視界が遮られ、

不定詞句が３つ。certa subsidia conlocari : 「整然と援軍を配置できず」

provideri quid esset opus in quaque parte: 「何が其々の地域に必要か見通す事が出来ず」(quid esset~は間接疑問文、名詞）

neque omnia imperia administrari ab uno : 「すべての司令が一人から出されることも出来ない」

軍団は展開したが、それは本来の軍の規則や秩序が要求する事によってと言うよりは、自然の場所の制約（下りの丘であった）と  時間の制約によってであった。こちらの少しあちらに少しと分散された軍団が敵に抵抗をした際、、前に言った厚い木々の壁に遮られて視界が悪かったので、きちんとした援軍の配置はできず、何が各々の場所で必要かを予測できず、更には全ての指令が一人の人間からでるということもできないという状況であった。
