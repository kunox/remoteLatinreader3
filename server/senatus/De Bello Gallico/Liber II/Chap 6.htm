Gallorum oppugnatio est haec : ガリア人の戦い方はこれ（次のよう）である。

ubi circumiecta multitudine hominum totis moenibus : circumiecta multutudineは独立奪格　「大勢の人を全ての城壁に配置して、」moenibus(walls)は与格、: circumで（その他もあり）始まる動詞は与格の間接目的語を持つ(NewLatin 187.3)

coepi (Activeのみの欠如動詞）　coepioの二つが紛らわしい、意味は同じ、coepiは能動のみなので、coepioは古代に使用され、この時期には全てcoepiが使われている。能動完了分詞がceptus、しかしcoepioの方は受動完了分詞がcoeptus ここではcoepiの能動完了分詞とかんがえられる、Jaciが受動不定詞なので、受動の不定詞をcoepiが支配するときは　coeptus est + 受動不定詞の形。coepitus est amari (NewLatin 133)

lapides jaci sunt coepi : 石が投げられること（対格＋受動不定詞）が始められる

undique in murum : 壁のすべての場所で

murusque est nudatus defensoribus : 城壁が守備隊から剥がされ（defensoribusは分離の奪格）

testudine facta portas sccedunt : 盾を頭上にかざして門に突進する。(testudoと言うのは盾を重なるように互いの頭の上において進む陣形の名前）

murumque subruunt. 城壁を壊す。

ガリアたちの戦い方はBelgaeも同じだが、つぎの様である。まず大勢の人員で城塞全てを取り囲みながら、全ての壁に対して石を投げ始める。そして城壁は守備隊から剥がされると、盾を頭上にかかげ城門に突撃し、城壁を壊すのである。
