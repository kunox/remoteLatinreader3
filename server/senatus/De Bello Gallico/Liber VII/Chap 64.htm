se fore contentum : contentumは形容詞contentusで「満足している」の意。

esse perfacile factu. factuは目的分詞奪格で「指定の奪格」として扱われる。uで終わる目的分詞は非常に限られている。（NewLatin 340.2)

esse perfecile factu prhibere. 「遮断することに関して容易である。」

modo ipsi corrumpant sua frumenta incendant aedificiaque. : modoはonly if でここの接続法は、「奨励の接続法」と呼ばれ[let's ~]と訳される。（AG439, NewLatin 274)

qua jactura familiaris rei videant, se conseui perpetuum imperium libertatemque.　quaは奪格で「それにより」の意

また彼は皆に数にして1万5000の騎兵を、素早く揃えるよう命令した。歩兵については、前から持っているものに満足しており、運命を試すようなことはしないし、戦線で戦うようなこともするつもりがないと言った。しかし騎兵に関して十分持つのは、ローマ軍を食料や飼料から遮断することが大変容易になるからである。平穏な気持ちで、穀物を腐らせ、建物を焼き払うようしようではないか。それらの見るであろう家族の財産の犠牲によって、我らに永遠の支配権と自由がもたらされるのである。
