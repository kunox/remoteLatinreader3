nec facile totum corpus cingeretur corona militum. corpusのところがopusのテキスト多し。corona militumは円陣の兵で取り囲むこと？「作業全部が容易に敵兵によって取り囲まれないように」

nostros destinatos operi : 作業に割り当てられた味方の兵　作業中の兵

残りの包囲要塞化はその塹壕から４００フィート下がったところで行なった。それは次のようなつもりからであった。そのような大きな間隔を擁することは必須であったからである。それは全ての作業が容易に敵兵によって囲まれないように、また、突然、夜にそれらの包囲施設にむかって敵が大勢で向かってくることがないように、また昼間に作業中の味方に向かって投げやりなどを投げることが可能でないようにである。

iorica : これはどうも銃眼を備えた低い壁。胸墻。

pinna : 同じく銃眼を備えた壁。胸壁。同じようなもののようである。

cerva : 直訳すると鹿であるが、ここでは先の二股になった小枝のようなものを指すらしい。

pluteus : 胸の高さの臨時の壁。移動可能。多くは木材、小枝などで作られる。ここではlorica, pinnaは実はこのpluteusを横に繋げて作ったものを指す。これらは下の部分が土（agger)で埋められて固定されているという構造らしい。

grandibus cervis : 結合の奪格。

quae distarent inter se LXXX pedes. quaeはturrisを指す。

これらに銃眼のある胸壁、胸の高さの壁を付く加え、それらにはこの移動可能な壁と地面にとの結合部に向かっている大きな角のような木の枝がついており、それが登ってくる敵を防いでいた。そしてこれらの作業で攻撃櫓、それは８０フィートの間隔で配置されていたが、を取り囲んだ。
