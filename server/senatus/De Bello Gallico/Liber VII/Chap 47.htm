id quod : quodで始まる名詞節での典型の一つ。（NewLatin299.1)

quod animo proposuerat. ここにはsibiが省略されていて、「自身に示した事」でなければおかしい？またanimoも難解でWalpoleによれば場所の奪格である由。意味は「心の中で」。こういう例は他にあるのだろうか？

Caesar iussit cani receptui. ここも曖昧。signumまたはtubamが省略されていて「退却のための信号を鳴らすように命じた」receptuiは目的の与格（NewLatin191.1)

自分で思っていたことが達成できたので、Caesarは退却の信号を鳴らすよう命じ、また第10軍団、彼らと一緒にいたのであるが、のSignaを直ちに停止した。

demissae : 分詞構文、nonnullaeを修飾。壁から降りてくる様。

quem constabat dixisse inter suos. constabatは非人称で、it is well knowedの意。宣言した、というニュアンス。quemはFabiusを指し、対格の主語、動詞はdixisse。「Fabiusが仲間のなかで言ったことはよく知られている」

se excitari Avaricensibus praemiis. Avaricensibusは複、奪格。Fabiusの言った内容で「自分はAvricumの褒賞によって目覚めた」

Lucius Fabius、彼は第8軍団の百人隊長であったが、その日仲間内で次のように宣言したことは皆が知っている。自分はAvaricumでの褒賞に興奮した。誰も（私より）前に壁を登ることをさせ無いであろう。3人の小隊の兵士を得て、彼らによって押し上げられて壁に登り、そのお返しで彼らを一人づつ壁に引き上げた。
