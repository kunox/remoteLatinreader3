Caesar audiebatur discessisse a Gergovia, 「Caesarは撤退したことが聞かれた」が逐語訳。

Galli confirmabant Caesarem contendisse. Gallia人はCaesarが向かたことを保証した。

interclusum, coactumはCaesarを修飾する完了分詞。分詞構文。

今や、CaesarがGergoviaから撤退したことが言われていたし、Aeduiの反乱やGalliaの成功裏の蜂起などの噂があり、Galla人は行軍できず河も渡れずで、食料難から止むを得ず属州に向かったことを確認した。

tanta rerum commutatione : 原因の奪格？「事態が大幅に異なっているので」

intellegebat aliud consilium capiendum sibi. 難解。intellegoという動詞は不定詞、または対格を目的語にもち「理解する」という意味。（completeLexicon)　逐語的には「彼自身に対する取られるべき異なる計画を理解した。」（capiendumがconsiliumを限定していると考える。）つまり「これまで通りではない新しい案が必要であると考えた」という事であろうか。

atque antea senserat : alius にはatque, ac etなどと共に使われ、「比較において」というニュアンスがある。(Charlton Lewis)  Vol 3 Chap28の1行目に　alia ratione ac reliquai Galli bellum gerere とある。「残りのガリアの戦いのやり方と比較して異なるやり方」

それにしてもanteaの前にquid, utのような語が必要に思えるが...

cogitabat neque iam ut~ sed ut~. 「～するのではなく～することを考えた」

そいてLabienusは状況が大きく変わったので、従来どおりではなく非常に異なる計画が必要であると知った。今や、どうにか戦闘を行い敵を攻撃することではなく、軍団を無傷でAgedincumに連れ帰ることを考え始めた。

Bellovaci instabant ex altera parte, Comulogenus tenebat alteram exercitu. : 「Bellovaciが一つのところから押してくるし、Camulogenusは他の地域を軍団により押さえている。」

maximum flumen distinebat legiones. : 大きな河が軍団を分断している。

実際、ガリアで最強の部族であるとの評判を持つBellovaciが一方から押してくるし、他方はCamulogenusが装備よく訓練された軍団によって支配している。更に大きな河により軍団を守備隊と装備一式から分断している。
