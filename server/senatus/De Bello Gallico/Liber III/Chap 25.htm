alii .... alii - : 「一方では...　他方では-」

telis coniectis : 独立奪格

vollo : 分離の奪格

auxiliaresque, quibus ... ：quibusは間接目的の与格

ab hostibus ... pugnarentur. 自動詞pugnoの受動で、非人称。敵からも、戦われた。

lapidis telisque subministrandis,  caespitibus comportandis : 動形容詞が奪格とともに使われ、動形容詞は手段を表す。（AG507)

decumana porta : 野営地の後方の門。前方の門はpraetoria portaと呼ばれた。decumana portaには商人などが入ってものを売っていた由。

一方では砦の溝を埋め、他方では沢山の槍を投げて敵を壁や砦から後退させた。予備軍、彼らにはCrassusは戦いに関してはあまり信頼していなかったのだが、石や槍を運び、芝生を砦の塀の方に運び込んだりして、戦闘員のように見せていた。同様に敵も確実に、恐れの色も見せず戦った。そして高いところから投げられる槍は味方に損害を及ぼした。砦を一周した騎兵はCrassusに報告した。後方の門は同じ堅牢さではなく、攻撃しやすいと。
