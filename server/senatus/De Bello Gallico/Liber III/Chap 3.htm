quo in consilio : このquoはよくわからない関係詞。英語だと接続詞＋指示詞（by + thta)をラテン語はどんどん関係詞を使うらしい。（NewLatin251.6) = in quo consilio : で前に言った軍議を指す。「その会議の中で」= in which counsil

cum ~ : 時の節を導くcumで、接続法が用いられていることから、その時の状況環境を表している事がわかる。（NewLatin288.1.b)

tantum repentini periculi : 部分の属格の用法の一つ。形容詞tantumの名詞的用法とともに「それだけたくさんの予期せぬ危機」(newLatin 201.2, AG3346.a)　accidissetの主語「沢山の危機が起こり」

praeter opinionem : 思っていたことと異なり、「思いもかけず」

loca completa conspicerentur : loca(複）が主語で、completaは完了分詞でlocaを修飾「満たされた場所」conspicerenturは受動態で「見られた」：　～で満たされた場所が見られた→　「その辺中で～があふれていた」

subsidio veniri : subsidio は目的の与格 veniriはvenio(自動詞）の受け身で非人称動詞として働く(NewLatin 138.4)それは来られる＝誰かが来る　possetが省略されている（possentから類推）として「誰も来ることができない」

commeatus supportari possent neque : 必要な物は補給されない

interclusis itineribus 独立奪格　「道筋が閉鎖されているため」

desperata salute : 独立奪格　「身の安全については絶望的で」

eijus modi = ejusmodi一語の副詞もあり。＝of this sort, of such kind 「性質の属格」と考えられる。内容はut以下で説明される。

non nullae sententiae dicebantur. : が骨子。「そういう意見も少なからず表明された」内容はut以下で述べられる。

impedimentis relictis : 独立奪格「重い装備を捨て」

erupione facta : 独立奪格「急襲をかけ」

isdem itineribus quibus eo pervenissent : 手段の奪格　来た道と同じ経路を使って

会議では次のような意見が少なくなかった。つまり、重い装備を捨て、突然の騒乱を起こし、来た道と同じ経路で、安全に向かって疾走しようと。というのも、想像だにできない大変な危機が起き、今や殆ど全ての高地には武装兵たちによってうめつくされ、如何なる援軍も来られない状況で、道を閉鎖されているため必要な物資の補給もできない状況であったからである。
