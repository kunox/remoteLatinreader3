aliqot de causis :　幾つかの理由で、原因で　id acciderat de alquot causis.「それは幾つかの理由から起きた」

belli renovandi : bellum renovandi(動名詞で戦争を再開すること）→　belli renovandi (動形容詞文）　bellumは中性なのでrenovandiのまま　（動形容詞構文）

legionisque opprimendae : legionem opprimendi →　legionis opprimendae (leginisはｆなのでdaeに）

Galli caperent consilium belli renovandi legionisque opprimendae : 「ガリア人達が、戦いを再開し軍団を抑えようという計画を思いついた事。」最初のidを説明している「結果の名詞節」(NewLatin 297)


理由は２つでこれから述べられるが、構成としては、quodで導かれる名詞節である。

　(primum) quod despiciebant legionem propter paucitatem,「何故なら少数の故に軍団を侮った」

　(tum etiam) quod existimabant ne posse sustineri propter iniquitatem loci「何故なら不利な場所のために支えきれないと考えた」


legionem, neque eam plenissimam, : 軍団、それも完全ではないもの、et ea という言い回しがあり「それも」の意。(NewLatin247.4)

detractis cohortibus duabus : 独立奪格　「２大隊が分離された」

compluribus (militibus) absentibus : 独立奪格　「多くの兵達が不在であり」

commeatus petendi : commeatus petendi 動形容詞文　「食料を調達するために」


tum etiam : ２つ目の理由の導入「かつまた」

cum ipsi : ガリア人のこと

ex montibus in vallem：　decurrerentとcoicerentの両方に掛かる。

suum posse sustineri : suumはローマ人のこと、対格で不定詞の主語

突然ガリア人達が、戦いを再開し、軍団を粉砕しようという計画を思いついたのには幾つかの理由があった。一つは、彼らが数の少なさのために侮ったことである。と言うのは軍団、それも不完全なものであり分離された二つの軍団であり、また多くの兵達が、食料の調達のためなどの理由で不在であったという事情があった。そしてまた、場所の悪さもあった。と言うのは彼らは山側から駆け下り、槍を投げれば、最初の一撃でさえ、我々が持ちこたえる可能性はないように思われた。
