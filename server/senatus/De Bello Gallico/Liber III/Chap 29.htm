incredibili celeritate : 「驚くべき速さを伴って」付帯状況の奪格か？

paucis diebus : 「短期間のうちに」これはLine1と同じで時の奪格（どの位）　NewLatin231

magno spatio confecto : 独立奪格　「大きなスペースが作られた」

extrema impedimenta : extremaは「終わり」という意味で、荷駄の最後尾部分の事

eius modi = ejusmodi(副詞）=of this sort, of this kind （Chap12,13にも出てくる）どういうものかは後のut節で記述される。ここでは　tempestates sunt consecutae で嵐があって、それがuti以下の様な事を引き起こす様な種類のものである、ということ。

pellibusは動物の皮のことで、野営テントはこれで出来ていた.

驚きべき速さで、短期間のうちに非常に広い場所が作られた。（しかし）味方によって、家畜や荷駄馬車の後尾が襲われても彼ら自身は更に森の深いところを目指すのであった。嵐が起きて、それはひどくて必要な作業が中断され、連続する長雨によって、兵士たちは安全にテントの中にいることができなくなった。
