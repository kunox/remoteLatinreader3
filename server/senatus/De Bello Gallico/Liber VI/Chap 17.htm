in multis civitatibus : 多くの国で、

licet conspicari tumulos exstructos harum rerum. : licet conspicari 見られることが許されていた。tumulos exstructosがconspicariの主語で、「積み重なった塊が見られることが許されていた」

consecratis locis : 場所の奪格。形容詞と一緒だと前置詞なしでもちられる。

accidit : 主語は続くut節。

neglecta religione : 独立奪格「信仰を無視して」

quispiam auderet aut occultare capta aut tollere posita. 「誰かが取得物を隠したり、置かれたものをとったりすること」

ei rei, gravissimum supplicium est constitutum. 「それらの事に対して最も厳しい懲罰が決められていた。」

多くの国では、これらの積み重なった塊が見られた。また余りよく起きたことではないが、誰かが信仰を無視して、取得物を自身の内に隠したり、置かれたものを盗んだりした場合には、最も重い懲罰が拷問とともに、決められていた。
