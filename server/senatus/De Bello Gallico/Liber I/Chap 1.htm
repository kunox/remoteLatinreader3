Gallia est omnis divisa in partes tres,: omnisは主格でGalliaを修飾。divisaは完了分詞の形容詞的用法。

quarum unam incolunt Belgae, : incoluntはincoloの３複現在でBelgae（３複）と一致。即ち、Belgaeが主語で動詞がincoloである。incoloは他動詞で、「～に生息する」というふうに目的語を取る。それがunam (partemが省略されている）である。quarumは前のpartesを受け（両方共に複数）副詞的に使われ、意味は「その中で」となる。直訳すると「その中で、ベルガエ族が一つの地域に住む」となる。

aliam Aquitani, これは前文のBelgaeの代わりにAquitani,　unam partemの代わりにaliam partem、という省略形。「Aquitaniが別ののところ」が逐語訳。

tertiam qui ipsorum lingua Celtae, nostra Galli appellantur. : ここはもうひとつの省略形がある。tertiam ei, qui という風にei(彼ら）が３番目に、と前文と同じ形式であるがそのeiは省略されて、eiを説明する関係代名詞文が後に続く。このような、eiを前に出して、それを後から関係詞で説明する、という形式は多い。逐語的には「自身の言語ではCaltae,我々の言語ではGalliと呼ぶ人たちが３番目の地域に住んでいる」

Belgae sunt fortissimi : ベルガエ族が最も強い。

horum omnium : omniumは形容詞omnisの複、属格で、horum (hicの複属格）と一致。「部分(全体の)の属格」と言われる用法で部分を表す語の後に全体を表す語の属格が続く(NewLatin201, AG346.a)。ここではちょっと変則で、fortissimi omniumという具合に使われている。「これらの全体の中で最も強い」

propterea quod : quodはこの場合接続詞でbecauseの意味がある。proptereaは副詞でtherefore。propterea quodはquodの強調形であり、意味は同じ。この表現はよく出てくるので一種の慣用句として覚える方が簡単。

次の文は　absunt a cultu atque humanitate provinciae. であり主語はBelgaeである。absuntは[ ab+sum ]で、～から（離れて）存在する、というニュアンス。逐語的には「（彼らは）属州の文化、人間性から離れていたので」

mercatores commeant minimeque saepe

     (atque) important ea quae ~ : 次の文はmercatores（商人たち）が主語で、commeant(訪れる)とimportant(運びこむ)の二つの動詞をもっており、２つともminime（最小限しか）という副詞にかかっている。「つまり商人たちは最小限の頻繁さで訪れ、最小限しか運び込まない」というのが大筋の文意。

最初の文ではad eos(=Belgae)彼らに向かって

importantも目的語は、eaでそれをquaeという関係代名詞文で説明している。

quae pertinent ad effeminandos animos: pertinentは関連するという意味の動詞で目的語はad以下。effeminandosはeffeminoの動名詞であり、ad+動名詞は目的を表す(AG506）逐語的には、「精神を弱める事に関連する」（もの）となる。

qui trans Rhenum incolunt : ここのincoluntは自動詞。incoloは前出の他動詞的な使い方とこの自動詞的な使い方の両方の意味がある。

quibuscum continenter bellum gerunt : quibuscum = quibus + cum。cumは奪格支配の前置詞でwithの意。他にはmecum(me + cim) secum(se + cum)などがある。quibusは関係代名詞quiの奪格でGermanisを指す。

その中で最も強いのはベルガエ人である。なぜなら、彼らは属州の文明、人道から最も離れたところでいて、商人たちも彼らのところにはめったに訪れず、軟弱になるようなものを取り入れなかったからである。また彼らは最もゲルマンに近く、彼らはレヌム河の向かいに住んでおり、彼らといつも戦争をしていた。
