videndum est : 「見られるべきである。」「見ることが必要だ」「考えるべきだ」

videndum est ne id sit non satis. : videoはutまたはneで受ける。neは～ではないかと心配する、というニュアンス。「それは十分では無いかもしれないと考えるべきである」

id, quod est dictum apud Platonem in philosophos. : idを説明する関係詞節。inはここではaboutの意。「プラトンによって、哲学者について語られた事」その内容は続くquod節によって述べられる。（内容的にはde Repub.の中で述べている、哲学者のみが正しく国を治められる、なぜなら彼らは世俗の野心、欲望から自由であるからである」と言うこと）

quod versentur in veri investigatione. : veriは目的語の属格。「（彼らは）真実の探求に生きており」

(que) quod contemnant et putent pro nihilo ea ~ : eaは続く関係詞節で説明されるが、それらを「見下し何でもないように考える」

propterea iustos esse. 「高潔であるがゆえに」

ea, quae plerique expetant vehementer, : 「大多数のものがそれを激しく渇望する」ようなもの。

(ea), de quibus inter se diglandiari soleant: 「それに関し、彼らの間で剣で争う傾向がある」（逐語訳）

それ故、プラトンが哲学者について言ったことは十分で無いかもしれないと考えるべきである。つまり、（哲学者は）真実の探求の中に生きており、高潔であるがゆえに、多くの人は激しく求め、剣を抜いて争うような事柄を見下し、何でもないように考える、と言うことである。
