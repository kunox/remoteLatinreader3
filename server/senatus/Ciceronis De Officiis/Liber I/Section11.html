ut ~ est tributum omni animantium generi. : generiは与格。すべての命あるものにut以下のことが備わっている。

１）ut se, vitam corpusque tueatur : 自身、命、身体を守る事

最初にいうが、すべての命あるものに対しては以下のようなことが自然により備わっているものである。一つは自身、命、体を守ること、また傷つけるであろうものを避ける事、さらに、生きるために必要なすべてのもの、それらは食料であり、住まいであり、その他のものであるが、を探し揃える事である。

hoc interest, quod ~ : quodは名詞節を導きhocと同格である。（NewLatin 299.1) 「quod以下の違いが存在する。」

tantum, quantum は２つとも同義。ラテン語によくある繰り返し。

quod節のなかに２つの動詞、movetur と accomodat seである。

accomodat se ad id solum quod adest que quod est praesens. ここのquodは関係代名詞でidが先行詞。「すぐ近くにあるもので、現在のもの」

paulum ~はsolumに対応して、accommodat se ad id paulum~. と思われる。

しかしながら、人と動物の間には大きな違いがある。動物はせいぜい、感覚によって動き回るだけであり、自分を身近にあるものまたは現在あるものにのみ対応させ、過去や未来の理解などは殆ど無い。

quod est rationis particeps: 「なぜなら、理屈の所有者であるので」「理性を授けられているので」

per quam : rationisを指し、「それを使って」

earumque praegressus et quasi antecessiones non ignorat. : praegressusが難しい。意味的にはantecessiones(前に起きたこと）に対して（今後起きること）という意味であろうが、主格しかない、ignoratの目的語とするには対格が必要。他方で、progressusとのMSSもあり、これだと対格があり、意味もadvanceで通じるか？

quasi = 謂わば、の意。antecessioが造語なのでその弁解風。

全体の構文としては、

Homo cernit consequentia, 「論理的帰結を理解し」

videt rerum causas : 「物事の原因を知り」

non ignorat earum praegressus et quasi antecessiones : 「それの来るべきものと起きたことの（因果関係を）無視せず」

comparat similitudines rebus : 「物事に類似性を適用し」

adjungit praesentibus : 「現在の状況に適応し」

adnectit futuras : 「将来に結びつけ」

videt totius vitae cursum : 「全人生のコースを見通し」

ad eam degendam praeparat res necessarias.: 「来るべきことに対して必要なものを用意する」

一方、人間は、理性を授かっているので、それを通して、物事の帰結を分けて、物事の原因を理解し、それらの来るべきものと起きたことを無視せず、物事の類似性を当てはめ、現在の状況に適応し将来に結びつけ、全人生の行く末を用意に見通し、来るべきにそなえ必要なものを用意する。
