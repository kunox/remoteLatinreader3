nostra legens : nostraは我々のものであるが、往々にして「私の」になる。（編集者のwe, NewLatin242.3)　ここでは私のを（文）を読むこと

nostra legens (est) non dissidentia a Peripateticis. : 私の文を読むことは、逍遥学派のものとあまり変わらない。

quoniam volumus esse utrique Socratici et Platonici. : 何故なら私は、ソクラテスやプラトン派でありたいと願っているからである。

utere tuo ipsi iudicio de rebus . utereは多分命令形で「その点については君は自身の判断を行いたまえ」

utere, uterisは２人称単数、現在と未来両方ある。かつ命令態にもある。←Holdenによれば、未来形でソフトな命令形を構成しているという。同様な仕様はCatil.に多く、patiere, abuntere, verebere などなど。

efficies orationem autem Latinam pleniorem legendis nostris. : legendis nostrisは手段の奪格。Latinam, pleniorem は述語対格で「ラテン語能力は強化される」

しかしながら、私の話を読むことは逍遙学派の話を読むこととそう大きく違わないであろう。というのは私彼らと私ははソクラテス派またはプラトン派でありたいと願っているからであるが、しかしこの点については君は自身の判断を行い給え。何もそれを邪魔することは無い。私の話を読むことで、明らかに君はラテン語の能力を強化することになるであろう。

nam philosophandi scientiam concedens multis: sumが省略されていて、「多くの人に哲学的知識において先行されているが」直訳だと「（私は）多くの人に対して哲学的知識を譲っているが」

si assumo id mihi, quod est ~「もし自分に対し以下のことを仮定すれば」　

quod est apte dicere : apteは副詞で、dicereは不定詞「適切に話をすること」

distincteは「如何に明確に、または適切な話法を使うか」

ornate は如何に適切な例えや修飾をするか

in eo studio consumpsi aetatem : 「その追求に私は一生を費やしてきた」

videor vindicare id meo jure 「自分の権利によってそれを要求できると思う。」

quodam modo : ある一定の範囲を伴って（随伴の奪格？）

哲学的知識という点では多くの人に先を譲っているが、もし以下のことを仮定すれば、自分も一定の範囲において、自身の優位性を要求する権利はあると思う。それはすなわち、演者の特性で、如何に適切に、かつ明確に、また適切な言葉の綾で話をするかという点である。それは私が全人生をかけて学んで来たことであるからである。
