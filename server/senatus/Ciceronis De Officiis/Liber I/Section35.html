bella sunt suscipienda ob eam causam: 「戦争はこのような理由から引き受けるべきである。」causamと続くutで始まる名詞節は同格。

ut sine iniuria pace vivantur. 「すなわち不正なしで平和に暮らせることである。」

parta victoria (sunt) conservandi ii : iはii = ei のテキスト多し。(i, iiはM複主格で同じ）eiは続く関係代名詞節で説明される。suntがないと動詞がなくなる。parta victoriaは独立奪格「勝利を得た場合には彼らは維持されるべきである。」

ut miores nostri : このutはasで「偉大な我らの祖先がしたように」

in civitatem etiam : 彼らの部族にいる状態で、すなわち、部族としてまるごと維持した、というニュアンスであろう。

Karthaginem = Carthaginem なぜKで始まるのか解らない。

nollem Corinthum (esse sublatam ) : 直前にsustuleruntと同じ単語があるので省略されたのであろう。「滅ぼされる事は望まなかった」

sed credo aliquid (esse) secutos : 間接話法に成っている。「何かが続いたと信じる」secorのニュアンスが難しい。secor = sequor

何かが（征服のあとに）起きた、何かがあった、ぐらいのニュアンスか？

ad locus faciendum bellum : 戦争を起こすべき立場、位置

それゆえ、戦争を始める理由は次のようであるべきである。つまり、不正なく平和に暮らせるということである。また勝利を得た場合には彼らは維持されるべきである。ちょうど我らの偉大な祖先が、Tusclanos, Aequos, Volscos, Sabinos, Hernicosなどを部族ごと受け入れたように。しかしCarthagoとNumantiamが根こそぎ滅ぼされたが。Corinthusについては私は滅ぼされたくはなかったが、何かの事情があったと思われる。それは彼らの非常に有利な場所、位置であり、彼らがいつかまた戦争を起こす気にならないように、滅ぼした。

iis consulendum est : 「それは彼らに面倒を見られるべきだ」行為者の与格で「彼らは保護されるべきである。」

quos deviceris vi : 彼らを（我々が）力により征服した。iisにかかる関係詞節。

tum ii(ei) (sunt) recipiendi. eiは複数、recipiendiは動形容詞で、叙述用法。（動名詞は主格はないし、複数形もない）

quamvis aries percusserit murum. : 「たとえ、破城槌が城壁を打っても」Romaの慣例として、破城槌が城壁を打つ前に降伏すれば助けるということがあった由。（BGII 32)

我らが征服した彼らは面倒を見られるべきであるのみならず、武器を置き、皇帝に忠誠を誓えば、たとえ破城槌が城壁にあたっても、彼らは保護されるべきである。
