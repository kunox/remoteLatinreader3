tum : next,

anquirunt : （自分で）考える　consultant (他の人にも）相談する。

構文的には、

tum anquirunt id conducat necne. 「それが正しいかどうか、それが（前のポイント）に資するものであるかどうか、考慮する」間接疑問文。

ad vitae commoditatem jucunditatemque : 人生の快適、楽しみに向かって（逐語訳）

ad facultates rerum atque copias : 諸事の方法と富

omnis cadit in rationem utilitatis. : 「全てはその有用性に帰する」

de quo deliberant, quae deliberatio : 構文は？  

de quo delibernt: これは繰り返しで、「それらについて考慮する」

quae deliberatio omnis : 先行詞がdeliberatioで、同格（前に述べた事と）の場合の先行詞に埋め込み（NewLatin251.4.(b))

次によく確認し、人にも相談すべきは、それが、人生の快適さ、楽しみに資するものか、また物事の手段や富をについて役立つか、はたまた、資産、影響力などそれを使って自身や友人たちの助けになるものかどうかなどである。つまり、すべては、その有用性に帰するものである。
